#  Fitness App

Androis app for physical exercise.

##  Technologies
- Kotlin
- Hilt
- Room
- Firebase
- Clean architecture
- Coroutines
- Navigation component
- jetpack compose

##  Possibilities
- Navigate between screens
- Make a list of exercises for each week
- Get more detailed information about the correct execution of each exercise
- Save exercises in a database
- Get tips for preparing for exercises
- Register in the app
- Login in the app
- Change password for login
